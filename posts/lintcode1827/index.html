题型

坐标DP + 方案总数

信息：坐标，步数

解决方案

  1. 状态： dp[i][j]表示走了i 次以后, 走到坐标j 的路径数
  2. 初始化：dp[0][0] = 1
  3. 方程：dp[i][j] = dp[i - 1][j - 1] + dp[i - 1][j] + dp[i - 1][j + 1]
    • 特殊情况：下标越界
  4. 答案: dp[steps][0]

Improvement: rolling array

代码

class Solution:
    """
    @param steps: steps you can move
    @param arr_len: the length of the array
    @return: Number of Ways to Stay in the Same Place After Some Steps
    """
    def num_ways(self, steps: int, arr_len: int) -> int:
        # write your code here
        if arr_len == 1: return 1
        mod = 10**9 + 7
        n = min(steps//2+1, arr_len)
        dp = [[0]*n for _ in range(2)]
        dp[0][0] = 1
        for i in range(1, steps+1):
            for j in range(n):
                dp[i%2][j] = dp[(i-1)%2][j]
                if j > 0:
                    dp[i%2][j] += dp[(i-1)%2][j-1]
                    dp[i%2][j] %= mod
                if j < n-1:
                    dp[i%2][j] += dp[(i-1)%2][j+1]
                    dp[i%2][j] %= mod
        return dp[steps%2][0]
