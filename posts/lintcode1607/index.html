https://www.lintcode.com/problem/1607/

题型

背包+ 方案总数

信息：项目数，人数，利润

解决方案

  1. 状态： dp[i][j][j]表示前i个项目(不需要所有项目都做)，用了j个人，获得至少k的收益的方案总数
  2. 初始化：dp[0][0][0] = 1, 注意：dp[0][0][i] = 0,如果i大于0
  3. 方程：dp[i][j][k] = dp[i-1][j][k]+dp[i-1][j-group[i-1]][k-profit[i-1]] ,其中group[i-1]代表第i个项目所需人数
  4. 答案: sum=(dp[N][j][P])，其中N = max(i), P = max(k), j in [0: people]

代码

from typing import (
    List,
)

class Solution:
    """
    @param g: The people in a gang.
    @param p: A profitable scheme any subset of these crimes that generates at least P profit.
    @param group: The i-th crime requires group[i] gang members to participate.
    @param profit: The i-th crime generates a profit[i].
    @return: Return how many schemes can be chosen.
    """
    def profitable_schemes(self, g: int, p: int, group: List[int], profit: List[int]) -> int:
        # Write your code here.
        n = len(group)
        dp = [[[0]*(p+1) for _ in range(g+1)] for _ in range(2)]
        dp[0][0][0] = 1
        for i in range(1, n + 1):
            for j in range(g+1):
                for k in range(p+1):
                    dp[i%2][j][k] = dp[(i-1)%2][j][k]
                    pk = max(0, k -profit[i-1])
                    if j >= group[i-1]:
                        dp[i%2][j][k] += dp[(i-1)%2][j-group[i-1]][pk]
        result = 0
        for j in range( g+1):
            result += dp[n%2][j][p]


        return result
