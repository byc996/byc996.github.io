Lintcode1869 · Count Square Submatrices with All Ones

题型

坐标DP + 方案总数

信息：坐标

解决方案

  1. 状态：dp[i][j]代表以(i, j)为右下角的全1正方形的个数(或最大的边长)
  2. 初始化
    • dp[i][0] = matrix[i][0]
    • dp[0][j] = matrix[0][j]
  3. 方程：dp[i][j] = dp[i-1][j]+dp[i][j-1]+dp[i-1][j-1]
  4. 答案: sum{dp[i][j]}，对任意i,j

Improvement: rolling array

代码

from typing import (
    List,
)

class Solution:
    """
    @param matrix: a matrix
    @return: return how many square submatrices have all ones
    """
    def count_squares(self, matrix: List[List[int]]) -> int:
        # write your code here
        n, m = len(matrix), len(matrix[0])
        dp = [[0]*m for _ in range(2)]
        num = 0
        for r in range(0, n):
            for c in range(0, m):
                if matrix[r][c] == 0: 
                    dp[r%2][c] = 0
                    continue
                elif (r==0 or c==0):
                    dp[r%2][c] = 1
                else:
                    dp[r%2][c] = min(dp[(r-1)%2][c-1], dp[(r-1)%2][c], dp[r%2][c-1])+1
                num += dp[r%2][c]
        return num
