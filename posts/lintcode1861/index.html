https://www.lintcode.com/problem/1861/

题型

坐标DP + 方案总数

信息：坐标，步数

难点：奇数步，偶数步；可以跳到-1，-2，-3

解决方案

  1. 状态：
    • dp[i][0]表示跳跃到i且最后一步为偶数步的方案数
    • dp[i][1]表示跳跃到i且最后一步为奇数步的方案数
  2. 初始化：dp[0][0] = 1
  3. 方程
    • dp[i][0] = dp[i-1][1]+dp[i-3][1]+dp[i-4][1]
    • dp[i][1] = dp[i-1][0]+dp[i-2][0]+dp[i-4][0]
  4. 答案: sum{dp[i][0 or 1]}，i=所有可能的倒数第一步的位置

Improvement: rolling array n->5

代码

from typing import (
    List,
)

class Solution:
    """
    @param arr: the steps whether have glue
    @return: the sum of the answers
    """
    def rat_jump(self, arr: List[int]) -> int:
        # Write your code here.

        # state: dp[i][0] - the number of ways jumping to i with the last step is even
        # dp[i][1] - the number of ways jumping to i with the last step is odd

        n = len(arr)
        rl = 5 # rolling length
        dp = [[0]*(2) for _ in range(rl)]
        # initialization
        dp[0][0] = 1
        odd = [1,2,4]
        even = [1,3,4]
        mod = 10**9+7
        for i in range(1, n-1):
            dp[i%rl][0] =  dp[i%rl][1] = 0  # 易错，使用前清空
            if arr[i] == 1: continue
            for jump in even:
                if i - jump >=0:
                    dp[i%rl][0] = (dp[i%rl][0]+dp[(i-jump)%rl][1])%mod
            for jump in odd:
                if i - jump >=0:
                    dp[i%rl][1] = (dp[i%rl][1]+dp[(i-jump)%rl][0])%mod
        ways = 0
        for jump in even:
            for i in range(max(0, n-1-jump), n-1):
                ways = (ways + dp[i%rl][1])%mod

        for jump in odd:
            for i in range(max(0, n-1-jump), n-1):
                ways = (ways + dp[i%rl][0])%mod
        return ways
